The HyperCSS Project

HCSS is a CSS framework. It helps you to create simple websites in a very short time and it increases your productivity.

Exemple : You created a software and quickly need a website to displays soime informations about it. But the most used frameworks like Bulma or Bootstrap are too large and your website will take a long time to load if you're using a CDN... Even if you use the local version of Bulma or BS, it'll take too many space!

So, I created the HCSS project, a basics-only and very lite CSS framework.

Made by The Fallen Angel, using HyperCSSv1.1.
